# AGENTS.md Template

新しい Codex プロジェクトを始めるときに、このリポジトリの `AGENTS.md` をプロジェクト直下へ配置して使います。

## 抽出元

複数の既存 Codex プロジェクトで使っていた `AGENTS.md` から、汎用的に使えそうな文言を抽出しました。

残した共通方針:

- UTF-8 でのファイル入出力。
- 日本語での説明。
- 変更前の関連ファイル確認。
- 小さく目的に集中した変更。
- 変更後の確認。
- `docs/` 配下へのドキュメント整理。
- 仕様変更時の `docs/specification.md` 同時更新。
- 既存の未コミット変更を上書きしないこと。
- 危険な操作や外部公開前の確認。

## 公開時の使い方

1. 新規プロジェクトでは `AGENTS.md` をそのまま取得してプロジェクト直下に置きます。
2. 必要なら「プロジェクト固有メモ」だけ追記します。
